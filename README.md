# Strapi v4 Firebase Authentication

This is a Strapi v4 thats use Firebase Authentication.

## How to use

1. Clone this repository

2. Install dependencies

```bash
npm install
```

3. Create a Firebase project

4. Create a Firebase service account

5. Add the service account json to the project and rename to serviceAccountKey.json

6. Start MySQL server on Docker or connect to existing MySQL server

```bash
docker-compose up -d
```

7. Start Strapi

```bash
npm run develop
```

## How to authenticate

Use firebase id token to authenticate
